## 接口设计

##### 1.登录和注册

###### 1.1 注册

| 接口 | api/register |
| ---- | :----------- |
| 描述 | 用户注册     |
| 方法 | post         |

**Request**

| 参数名 | 类型   | 是否必须 |
| ------ | ------ | -------- |
| user   | Object | Y        |

**Response**

| 参数名     | 类型   | 是否必须 |
| ---------- | ------ | -------- |
| error_code | int    | Y        |
| error_desc | String | Y        |

###### 1.2 用户登录

| 接口 | api/login |
| ---- | :-------- |
| 描述 | 用户登录  |
| 方法 | post      |

**Request**

| 参数名   | 类型   | 是否必须 |
| -------- | ------ | -------- |
| id       | int    | Y        |
| password | string | Y        |

**Response**

| 参数名     | 类型   | 是否必须 |
| ---------- | ------ | -------- |
| uid        | int    | Y        |

| error_code | int    | Y        |
